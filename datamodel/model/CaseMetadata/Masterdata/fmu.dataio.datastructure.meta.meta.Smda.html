Smda

StratigraphicColumn

Reference to stratigraphic column known to SMDA.

CountryItem

Reference to a country known to SMDA.

CoordinateSystem

Reference to coordinate system known to SMDA.

FieldItem

Reference to a field known to SMDA.

DiscoveryItem

Reference to a discovery known to SMDA.
pydantic model Smda[source]

Block containing SMDA-related attributes.

Fields:

  • coordinate_system (fmu.dataio.datastructure.meta.meta.CoordinateSystem)

  • country (List[fmu.dataio.datastructure.meta.meta.CountryItem])

  • discovery (List[fmu.dataio.datastructure.meta.meta.DiscoveryItem])

  • field (List[fmu.dataio.datastructure.meta.meta.FieldItem])

  • stratigraphic_column (fmu.dataio.datastructure.meta.meta.StratigraphicColumn)

field coordinate_system: CoordinateSystem [Required]

Reference to coordinate system known to SMDA. See CoordinateSystem.

field country: List[CountryItem] [Required]

A list referring to countries known to SMDA. First item is primary. See CountryItem.

field discovery: List[DiscoveryItem] [Required]

A list referring to discoveries known to SMDA. First item is primary. See DiscoveryItem.

field field: List[FieldItem] [Required]

A list referring to fields known to SMDA. First item is primary. See FieldItem.

field stratigraphic_column: StratigraphicColumn [Required]

Reference to stratigraphic column known to SMDA. See StratigraphicColumn.
