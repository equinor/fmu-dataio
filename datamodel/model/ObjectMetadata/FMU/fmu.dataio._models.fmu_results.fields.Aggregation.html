Aggregation

Parameters

The parameters block contains the parameters used in a realization.
pydantic model Aggregation[source]

The fmu.aggregation block contains information about an aggregation performed over an ensemble.

Fields:

  • id (uuid.UUID)

  • operation (str)

  • parameters (fmu.dataio._models.fmu_results.fields.Parameters | None)

  • realization_ids (List[int])

field id: UUID [Required]

The unique identifier of an aggregation.

field operation: str [Required]

A string representing the type of aggregation performed.

field parameters: Optional[Parameters] = None

Parameters for this realization. See Parameters.

field realization_ids: List[int] [Required]

An array of realization ids included in this aggregation.
