pydantic model AnyData[source]

The data block contains information about the data contained in this object. This class, AnyData, is a root model that allows for data with more specific content types to be placed within it. It can contain the metadata for any data object.

See Data to get an overview of all of the subfields used in the data block. Between the different content types, only the data.content field will differ. This field indicates the type of content the data are representing.
